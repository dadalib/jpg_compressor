## Getting Started

Intall your python and pip  .
* Python 3.9.2
* pip 22.0.3


### Prerequisites

All requires library are present in requirements.txt  file.
* Pillow
  ```
  pip  install Pillow
  ```

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:dadalib/jpg_compressor.git
   ```
2. Install  packages
   ```sh
   pip install -r requirements.txt
   ```
